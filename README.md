## 가상현실 환경에서 키워드 및 이미지 기반 LLM을 활용한 작업 지원

# 연구 목표
타이핑이 어려운 가상현실 환경에서 LLM 이용을 위한 효과적인 텍스트 완성 및 사전 프롬프팅 기능 제시 

키워드 기반 질문 생성을 통해 적은 노력으로 완성된 문장을 생성

텍스트와 이미지를 프롬프트로 활용해 사용자의 작업을 돕는 LLM 어시스턴스 제안

사용자가 혼자 작업을 하는 상황에서 작업 과정에 문제가 생겨도  타인의 도움 없이 효과적으로 상황을 해결할 수 있는 연구 필요

LLM에게 현재 작업 상태와 문제점을 효과적으로 설명할 수 있는 시스템 


# 실험과정
사용자가 가상환경상황에서 소마큐브를 조립하고 있는 상황에서 다음 과정으로 넘어가는데 문제가 있을 경우 컨트롤러를 이용해 조립도에 대해 정보를 부여받은 LLM에게 도움을 받을 수 있음
